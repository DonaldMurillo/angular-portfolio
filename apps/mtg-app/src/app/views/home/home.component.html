Welcome to MTG App

This prototype will let you:

  • Search Magic the gathering cards using the Scryfall api.
  • Create an account and login. (This is a prototype so only basic security was provided for the app, keep that in mind!)
  • Once you create an account, you can create collections.
  • If you are logged in while searching d=fir cards, you will be able to add those cards to your collections.
  • If the collection is tradeable, it can be shared with anyone using a simple link.
  • Light and Dark mode!

If you want to know more about how this app was made please visit the github repository

This app is a prototype and might experience feature rework, removal or anything without any warning. The database might (and probably will) be deleted from time to time.
Any feedback is welcome in the github repository.
