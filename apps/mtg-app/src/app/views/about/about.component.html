About

The purpose of this prototype was to showcase common Web Apps functionalities such:

  • CRUD
  • Authentication/Autorization
  • Web and mobile view (mobile first)
  • Third party API consumption
  • Custom API development

Stack

This app was made with Angular 13, Nestjs 8, Postgres 13 and Github as project management tool.
